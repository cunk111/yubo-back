# yubo-back
## Installation

### Prerequisite:
To install, your machine needs the following requirements
```
- yarn ^5.4.0
- npm 42^
- node ^10
```
To install, you need to perform

```
$ mkdir tmp && cd $_
$ git clone https://github.com/cunk111/yubo-back.git
$ cd yubo-back/
$ yarn run install
```

The project also expects a file `db.sqlite` at project root
